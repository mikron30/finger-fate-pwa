# Finger Fate – אפליקציית בחירת אצבעות

אפליקציית PWA לטלפון, בעברית, שפועלת במסך מלא וניתנת להתקנה ממסך הבית.

## חוק השעות

המחזור הבא חוזר כל 6 שעות לאורך כל היממה:

- 00:00–01:00: הראשון שמניח אצבע
- 01:00–02:00: ימין־למטה
- 02:00–03:00: שמאל־למטה
- 03:00–04:00: שמאל־למעלה
- 04:00–05:00: ימין־למעלה
- 05:00–06:00: האחרון שמניח אצבע

לאחר מכן אותו סדר חוזר בשעות 06:00–12:00, 12:00–18:00 ו־18:00–24:00.

## הפעלה מקומית במחשב

PWA ושירות לא־מקוון חייבים שרת מקומי, ולא פתיחה ישירה של index.html.

```bash
cd finger_fate_pwa
npx serve .
```

לאחר מכן פתחו את הכתובת שמופיעה במסוף.

## התקנה בטלפון Android

1. העלו את התיקייה לאירוח HTTPS, למשל GitHub Pages, Netlify או Firebase Hosting.
2. פתחו את האתר ב־Chrome בטלפון.
3. לחצו על "התקן כאפליקציה בטלפון" או בתפריט Chrome על "הוספה למסך הבית".

## קבצים מרכזיים

- `index.html` – מבנה המסכים.
- `styles.css` – העיצוב.
- `app.js` – זיהוי האצבעות וכללי הזכייה.
- `manifest.webmanifest` ו־`sw.js` – התקנה ועבודה לא־מקוונת.

הסמל והממשק מקוריים ואינם העתק של Chwazi.
