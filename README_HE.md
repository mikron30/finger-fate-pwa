# Finger Fate 2 – אפליקציית בחירת אצבעות

האפליקציה נפתחת ישירות במסך המשחק השחור.

## הפעלה

1. מניחים לפחות שתי אצבעות על המסך.
2. משאירים אותן במקומן עד הבחירה.
3. בסיום כל המסך נצבע בצבע האצבע הזוכה.
4. מרימים את כל האצבעות כדי להתחיל סבב חדש.

## פתיחת ההגדרות הנסתרות

בתחילת סבב, מניחים **אצבע אחת בלבד** ומשאירים אותה על המסך במשך **5 שניות**. לאחר מכן נפתח מסך ההגדרות המלא.

## חוק השעות

המחזור הבא חוזר כל 6 שעות לאורך כל היממה:

- 00:00–01:00: הראשון שמניח אצבע
- 01:00–02:00: ימין־למטה
- 02:00–03:00: שמאל־למטה
- 03:00–04:00: שמאל־למעלה
- 04:00–05:00: ימין־למעלה
- 05:00–06:00: האחרון שמניח אצבע

לאחר מכן אותו סדר חוזר בשעות 06:00–12:00, 12:00–18:00 ו־18:00–24:00.

## עדכון ב-GitHub Pages

יש להעלות מחדש את הקבצים שבשורש המאגר ואת תיקיית `icons`. הקבצים החשובים לעדכון הם:

- `index.html`
- `app.js`
- `styles.css`
- `sw.js`
- `manifest.webmanifest`
- `README_HE.md`

לאחר ה-Commit יש לסגור את האפליקציה המותקנת ולפתוח אותה מחדש. במידת הצורך, רעננו פעם אחת את האתר בדפדפן כדי לנקות את הגרסה הקודמת מהמטמון.
